# TeamTalk pro+ client application for Android

особенности

# защита от спама

этот клиент автоматический выключает подписку на спамеров. всё это можно настроить в настройках приложения:
лимит сообщений после которых будет выключаться подписка, возможность включить и отключить режим защиты, а также настройка отключить все подписки на пользователя

# звуковые темы

в клиенте есть возможность настроить какой звук будет играть при в каком-либо событии. а также есть режим без звука и дефолтная Тема которая включена по умолчанию

# администрирование

работает так же как на ПК:
редактирование учётных записей, управление забаненными пользователями, изменение свойств сервера, статистика сервера

# трансляция медиафайлов

переработано меню трансляция медиафайлов. теперь можно останавливать и заново запускать трансляция видеофайлов при этом не теряя положение в хронометраже аудио

# прочее

почти все экраны переписаны, также Тёмная тема выглядит намного лучше Старый
